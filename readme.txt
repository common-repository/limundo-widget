== Description ==

Limundo widget is a plugin that displays your auction from Serbian auction web site Limundo.com in your post/page.

== Installation ==

1. Upload the folder `limundo-widget` to the `/wp-content/plugins/` directory
2. Activate the plugin `Limundo` through the 'Plugins' menu in WordPress
3. In preferens menu chek setup page and configure Limundo.com and where to apply change.
